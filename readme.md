# Feature Engineering, Data Exploration and Machine Learning
The key idea behind this project is gaining practical experience in data analysis and feature engineering.
In addition to that I want to explore how classical machine learnings methods and neural networks performs in 
regression tasks.


## Project is based on data set from this competition
[House Prices - Advanced Regression Techniques](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/overview)
